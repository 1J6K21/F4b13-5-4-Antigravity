# RunwayTracker

A fully functional, interactive React and Tailwind web application for tracking startup runway, burn rate, and expenses.

## Features
- **Dashboard**: Track Current Cash, Monthly Burn Rate, and Estimated Runway in months.
- **Cash Projection Chart**: Visual area chart mapping out cash depletion over time.
- **Expense Manager**: Add and remove monthly expenses with dynamic category tagging.

## Tech Stack
- React 18
- Vite
- Tailwind CSS v4
- Recharts
- Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
