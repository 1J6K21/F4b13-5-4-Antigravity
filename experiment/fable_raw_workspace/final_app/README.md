# Nexus Intelligence

A Palantir/Apple-inspired data intelligence platform demonstrating advanced frontend architecture.

## Tech Stack
- React 19
- Tailwind CSS v4
- Three.js & React Three Fiber (3D Neural Globe with Bloom Post-processing)
- D3.js (Force-directed network visualization)
- Framer Motion (Layout animations and micro-interactions)
- GSAP (Advanced sequenced text animations)
- Lucide React (Icons)

## Features
- **3D Neural Globe:** Interactive, glowing particle system with line connections representing global intelligence data.
- **Topological Map:** Real-time force-directed graph built with D3.js for visualizing network integrity and nodes.
- **AI Terminal:** GSAP-powered terminal interface with sequenced multi-line typing animations.
- **Glassmorphism UI:** Heavy use of backdrop-blur, subtle borders, and neon glows inspired by Palantir & Apple WWDC presentations.
- **Live Metrics:** Simulated live data flow, processing latency, and anomaly alerts.

## Run Locally
```bash
npm install
npm run dev
```
