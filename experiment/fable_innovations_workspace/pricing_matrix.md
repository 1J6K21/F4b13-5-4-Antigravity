# DocuMind AI - Pricing & Feature Tier Matrix

DocuMind AI provides enterprise-grade document extraction, semantic analysis, and automated workflows.

## Pricing Architecture
The pricing model uses a **Hybrid Structure**: a base platform fee (for access, security, and core features) plus a usage-based component (per document or per 1,000 pages processed) to align with value delivery.

| Feature Category | Starter (Ad-hoc / Small Team) | Professional (Mid-Market) | Enterprise (Large Corp / Regulated) |
| :--- | :--- | :--- | :--- |
| **Target Audience** | SMBs, Boutiques | Mid-market, Growth | Fortune 500, Legal, Finance |
| **Base Platform Fee** | $499 / month | $1,999 / month | Custom (Starts at $5k/mo) |
| **Included Volume** | 1,000 pages / mo | 10,000 pages / mo | 100,000+ pages / mo |
| **Overage Rate** | $0.15 / page | $0.08 / page | Custom volume discounts |
| **Core AI Capabilities** | Standard OCR, basic entity extraction, summarization | Advanced semantic extraction, anomaly detection, multi-doc comparison | Custom model fine-tuning, domain-specific taxonomy generation |
| **Supported Formats** | PDF, DOCX, Images (Standard) | Standard + Handwritten, Scanned tables, Emails | All formats + legacy mainframes, audio/video transcripts |
| **Integrations** | Google Drive, Dropbox, Slack | Salesforce, Workday, REST API access | Custom ERP (SAP, Oracle), Webhooks, On-prem syncing |
| **Security & Compliance** | In-transit encryption | SOC2 Type II, GDPR, SSO (SAML) | HIPAA, FedRAMP, VPC Deployment, BYO-Key |
| **Support & SLAs** | Email support (48hr SLA) | Priority support (12hr SLA), Shared Slack channel | 24/7 Phone, Dedicated Customer Success Manager, 99.99% Uptime |

### Add-ons & Expansion Revenue
- **Human-in-the-Loop (HITL) Verification:** $0.50 per document flagged for low confidence.
- **Custom Template Training:** $1,500 one-time fee per complex document template.
- **Data Retention Extension:** $200/mo for 7-year audit-ready storage.
