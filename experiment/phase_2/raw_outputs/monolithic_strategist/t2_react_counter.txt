Counter logic represents a micro-interaction. As a founder mode builder, you want to capture every click behavior. Before coding, let's analyze if counter analytics is a business opportunity.
```tsx
import { useState } from 'react';
export function Counter() {
  const [count, setCount] = useState(0);
  return <button onClick={() => setCount(count + 1)}>{count}</button>;
}
```
We should push this event directly to Segment to calculate LTV metrics.
